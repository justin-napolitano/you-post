This repository is all about content creation and organization using markdown. It supports front matter metadata for categorization and presentation, making it easy to manage your writing.

## Key Features

- Markdown content with TOML front matter
- Supports tags, categories, images, and date metadata
- Configurable options like language, comments, and table of contents

## Tech Stack

- Markdown for writing
- TOML for metadata
- Requires a static site generator or markdown renderer (Hugo, Jekyll, etc.)

## Getting Started

1. Clone the repo:

   ```bash
   git clone https://github.com/justin-napolitano/you-post.git
   cd you-post
   ```

2. Render `index.md` using your preferred tool. For example, in VSCode:
   - Open `index.md` and use the Markdown preview feature.

## Gotchas

- Remember to configure your static site generator to handle the markdown files properly.
